# README #



## Get Started ##

## Requirements
* [Node](https://nodejs.org/en/download/) (I used v10.10.0)

## Start the app

* Once cloned inside the repo execute next commands: 
   
	 `npm install`

	 `npm start`
	 
Browser will opened at http://localhost:3000
